## 给原版玩家
### 综合修改
1. 鼓励进攻，增加战斗时间占比
   * 提高补给速度，步兵0.3->0.35，车辆0.15->0.2，坦克0.1->0.15，动员兵为0.9 (以0.35为例，1/0.35向上取整为3，即一个步兵从0回到满需要3次，所有单位的每次间隔时间是相同的)
   * 降低溃退士气，步兵20->15，车辆20->15，坦克15->10
   * 快速补给线，DS改为任意兵营可出，花费5，但不提供油料弹药
   * 提高将军经验获取速率，团队参战时速率1->2，单次获取上限10->50
   * 增加将军技能的选择，每个将军固定拥有借款(短时间的1000)和空投补给，cd时间为200
   * 取消快速第二主机技能，主机需要研究科技后从发动机车间花钱出
   * 支持进攻方拥有视野优势并打出先手，侦察类兵种可在地上插眼，花费5，不消耗弹药油料，视野范围比BT射程远，
   * 禁止了除补给站(DS)外所有建筑的展开后再打包，降低了兵营主机的防御，并适当调整血量，增加了步兵和机枪车对建筑的DPS
   * 进攻奖励，兵营、主机、工事、科技建筑打掉均有金钱奖励
   * 禁止了侦察兵(25)和日本突击兵(35)的出兵满编，仅允许动员兵出兵满编
   * 禁止了除侦察兵、步兵(50)、动员兵、机械化步兵(125)外所有团队的堑壕

2. 扩大规模，支持大军团对战，支持更多的操作性
   * 开放单位人口上限为30
   * 增加了大部分后排单位的射程，并允许他们轰击(B)
   * 所有坦克支持轰击，大幅提升了所有炮塔类单位的炮塔转速
   * 地图选择支持无限模式，将给所有玩家无需资源支持特性，允许无视单位人口上限出兵

3. 经济调整，苏德新经济模式
   * 苏德采用与美日英不同的经济模式，兵营不再提供金钱，改为需要出专门的经济建筑提供，并适当调整造价
   * 取消了弹药库油料库占用的建筑人口；针对苏德少兵营的情况，苏德弹药库油料库提供资源双倍
   * 针对苏德少兵营的情况，苏德兵营不占建筑人口，不需要主机提供补给，可出的单位最少为9最多为12 (UI显示上仍只能显示6个，实际数字以兵营提供的单位人口为准)
   * 针对苏德少兵营的情况，苏德步兵伞兵的招募速度是其他国家的接近3倍，车辆和坦克为2倍
   * 允许伞兵营(DB)出侦察兵，英国机械营(DZ)以伞兵营作为前置；针对苏德少兵营的情况，苏德的机械营无需步兵营(DT)作为前置

4. 增加玩法，降低多样化玩法的成本
   * 降低伞兵营单位空降的花费，空投下来的伞兵自带补给(日本敢死除外)且视野扩大；但相对的，击杀空投的伞兵会有金钱奖励
   * 降低机场的花费和轰炸的花费，机场前置科技仅需要工程车间，但相对的，加强防空单位，且击杀飞机(包括运输机)会有金钱奖励
   * 取消工兵，所有工事由科技建筑以卡车形式部署，并支持快速展开
   * 工事除了自动的防御武器，还拥有一个数据相同但允许主动轰击(B)的主武器，允许自行选择攻击目标
   * 工程车间允许出维修站，花费100对己方所有建筑以每秒25的速度回复血量，战斗状态的建筑无效，持续100秒后结束
   * 随机地图的简单普通困难模式可以自定义开局拥有的工事卡车数量
   * 新的坦克大战模式

### 单位修改

1. 步兵伞兵
   * 所有步兵增加下蹲和匍匐减伤，分别增加除了火焰外的全抗性5%与10%
   * 侦察兵取消招募满编，允许插眼
   * 步兵团增加了机枪手的射程，分散队形
   * 美德可在步兵营直接出机械化步兵，且改前排3个位置为3个半履带装甲车，可以有效阻挡狙击单位的首轮攻击
   * 反坦克步兵增加射程，攻击为AOE，允许轰击，对中坦以下车辆单位造成瘫痪，对其余单位造成减速，持续2秒，对建筑无效果
   * 迫击炮增加射程，士气打击仅限步兵且数值减少，但对建筑有50爆炸增伤
   * 防空步兵团配备3辆防空车
   * 日本突击兵团取消初始满编和无视地形限制，狙击去掉手雷，和苏联狙击一样允许轰击
   * 敢死步兵团的敢死兵改为6个且移到前方，后排增加两个机枪手
   * 动员兵需要动员令，初始满编
   * 美国枪榴弹步兵允许轰击
   * 日本可在主机研究小坦克科技并直接在步兵营出小坦克
   * 伞兵增加两个伞兵机枪手，重伞改为两个迫击炮与一个机枪手
   * 敢死伞兵移到前排且增加到4个
   * 去除了工兵团和喷火兵团

2. 机械化
   * 机械化装甲团为侦察兵上位，允许插眼同时可以修建筑
   * 机械化防空团增加到6个车
   * 去除了半履带反坦克团，允许出轻型坦克团，允许轰击，对中坦以下车辆单位造成瘫痪，对其余单位造成减速，持续1秒，对建筑无效果
   * 半履带自行炮增加射程，士气打击仅限非坦克且数值减少，但对建筑有50爆炸增伤
   * 研究狙击步枪后允许出半履带猎兵团，对步兵快速杀伤
   * 研究喷火器后允许出半履带喷火团，无视地形限制快速移动，自动攻击，对远程火炮单位施加高额伤害debuff，对其余单位伤害轻微，但可以显著降低其补给速度和士气恢复速度
   * 歌莉娅必须在机械化营出，改为机械化装甲团加两个歌莉娅车，歌莉娅不会被狙击秒杀，且增加了车辆抗性
   * 喀秋莎改为与半履带自行炮相同队形，且增加了其射程

4. 坦克
   * 所有坦克均增加为4辆
   * 增加坦克歼击车团，苏日4辆，其余国家3辆，允许较远射程轰击，对重坦以下车辆单位造成瘫痪，对其余单位造成减速，持续2秒，对建筑无效果
   * 坦克榴弹炮增加射程，对建筑有50爆炸增伤，日本为4辆，其余国家3辆
   * 重型坦克攻击时会回复士气
   * 喷火坦克改为半履带喷火车的上位
   * 取消了坦克喀秋莎
   * 修复了原版苏联所有坦克缺少加速度的问题

5. 飞机
   * 所有国家均可以研究凝固汽油弹

6. 建筑
   * 高级防御工事的前置科技为“改进型防御工事”
   * 修复了原版苏联碉堡射程短的问题

### 游戏模式
1. 随机地图
   * 除了默认的一个补给站外，苏德开局会送一个经济中心，其他国家则替换为补给站
   * 随机地图可以选择难度，工事均以卡车形式存在
     * 无限：拥有所有开局工事以及科技“无需资源支持”用以突破单位人口的30上限
     * 简单：拥有所有开局工事
     * 普通：拥有除火炮掩体外的开局工事
     * 困难：仅拥有机枪掩体
  
3. 脚本地图
   * 脚本地图支持无限模式
   * 有专门的地图支持合作打AI，联机团战，联机混战，脚本地图库持续更新中

4. 坦克大战
   * 在此模式下，美国会被替换为德国，英日会被替换为苏联
   * 除了开局赠送的130+经济，主机提供的25/30+经济，抢城市10/15+外，无额外增加经济手段
   * 出兵建筑仅限坦克营，造价150，无科技前置
   * 坦克营允许出半履带装甲团和半履带喷火团
   * 所有坦克和机械化的单个单位和团队击杀后均可获得金钱奖励，地图上除了城市外会生成大量初始野怪坦克团以供击杀
   * 允许出工事和额外主机
